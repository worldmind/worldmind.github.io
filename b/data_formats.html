Форматы данных

Для публикации данных государство должно использовать только свободные и стандартные форматы данных, тем самым гарантируя их доступность для всех граждан. В идеале на территории государства всё ПО должно использовать свободные и стандартные форматы данных, т. к. это стимулирует конкуренцию и защищает права пользователей на их данные (см. «Индивиды имеют право на приватность»), но маловероятно, что есть основания для полного запрета нестандартной продукции.

См. также «Защита интересов потребителей».
