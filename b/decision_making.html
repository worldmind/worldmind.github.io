Система поддержки принятия решений

Для коллективного принятия общественных решений необходим инструмент, который позволяет:

  1. выдвигать вопрос на всеобщее обсуждение и голосование;

  2. обсуждать этот вопрос для поиска консенсуса (см. «Процедура референдума») (хотя обсуждение может быть реализовано отдельным механизмом с целью упростить, а значит, сделать более надёжной систему голосования);

  3. голосовать по наиболее подходящему в каждом случае алгоритму (см. «Алгоритм голосования») и отзывать голос;

  4. выбирать случайных граждан для задач вроде суда присяжных, гражданской ассамблеи / жюри.
